# bitbank SMA Cross Trading Bot Sample

## 概要

これは、日本の暗号資産取引所 **bitbank** の公開APIを使用して、単純移動平均線（SMA）のクロスオーバー戦略に基づいた売買シグナルを検出する取引botのサンプルプロジェクトです。

このbotは、指定された通貨ペア（デフォルトは `btc_jpy`）のローソク足データを定期的に取得し、短期SMAと長期SMAのクロスを監視します。

**重要:** このサンプルは教育目的のものです。**実際の取引注文は行いません。** シグナルを検知すると、コンソールにメッセージを表示するだけです。

## 主な機能

-   bitbankから指定した通貨ペアの価格データ（ローソク足）を取得
-   短期および長期の単純移動平均線（SMA）を計算
-   ゴールデンクロス（買いシグナル）とデッドクロス（売りシグナル）を検出
-   設定ファイル (`config.py`) による簡単なカスタマイズ

## 必要なもの

-   Python 3.x

## セットアップ手順

1.  **依存ライブラリのインストール**
    このプロジェクトに必要なライブラリをインストールします。

    ```bash
    pip install -r requirements.txt
    ```

## 設定方法

botの動作は `config.py` ファイルで設定できます。

-   `PAIR`: 取引する通貨ペア (例: `'btc_jpy'`, `'xrp_jpy'`)
-   `CANDLE_TYPE`: 使用するローソク足の時間軸 (例: `'1min'`, `'5min'`, `'1hour'`)
-   `SHORT_SMA_PERIOD`: 短期SMAの期間
-   `LONG_SMA_PERIOD`: 長期SMAの期間
-   `INTERVAL_SECONDS`: データ取得とシグナルチェックを行う間隔（秒）

## 実行方法

以下のコマンドでbotを起動します。

```bash
python main.py
```

botを停止するには、`Ctrl+C` を押してください。

### 実際の取引を行うには？

実際の取引注文を行うには、**bitbankのPrivate API**を使用する必要があります。主な改造点は以下の通りです。

1.  **APIキーの取得**: bitbank公式サイトでAPIキー（API KeyとAPI Secret）を発行します。
2.  **APIラッパーの拡張**: Private APIを呼び出すための関数（注文、残高取得など）を `bitbank_api.py` に追加します。これには、リクエストの署名生成処理が必要です。
3.  **注文ロジックの追加**: `main.py` 内で 'buy' または 'sell' シグナルが検出された際に、実際に注文を出す処理を実装します。
4.  **リスク管理**: 注文サイズ（ロット数）の決定や、損切り（ストップロス）などのリスク管理ロジックを追加することが強く推奨されます。
